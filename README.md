## 1. Install libraries
```
pip install numpy
pip install turtle
```

## 2. Input
Change input in file input.txt  


## 3. Run searching algorithms
- BFS
```
python3 run.py --algorithm bfs
```

- UCS
```
python3 run.py --algorithm ucs
```

- IDS
```
python3 run.py --algorithm ids
```

- BGFS
```
python3 run.py --algorithm gbfs
```

- A* 
```
python3 run.py --algorithm astar
```
